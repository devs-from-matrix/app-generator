## How to build ?

- `npm i -g yarn`
- `yarn install`

## How to start ?

### To run all tasks you can use

`gulp --d [domain-name] --t [template-name] --go [GitHub Organization] --gt [GitHub Token]` 

### Help

`gulp helpme`
